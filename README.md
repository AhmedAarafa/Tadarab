
## Getting Started

First, install packages used:


REQUIRED:
```bash
npm version : 6.14.16
```

```bash
node version: 12.22.10
```


```bash
npm install
```
Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
